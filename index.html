Fire vs Water

Description

Un volcan est entré en éruption. Eviter les boules de feu et ramasser les gouttes d'eau pour calmer le volcan.

Comment jouer ?

  • Utiliser les flèches ⬅ et ➡ pour se déplacer
  • Attraper les gouttes d'eau pour maigrir, accélérer et augmenter votre score
  • Eviter les boules de feu qui vous font grossir, ralentir et diminuer votre score de 2 points
  • Appuyer sur la touche entrée pour lancer le jeu
  • Le jeu commence par défaut à 3 projectiles, mais vous pouvez le changer par la suite
